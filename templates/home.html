What does it do?

API Buf.fr was born when i encountered a problem with some web-based pseudo-RESTful API's. Stability. Suffice to say, i mistakinly overloaded a public API during development i had added a buffer. For the API. An API Buf.fr :D

What do i need?

API Buf.fr is fairly simply to setup, but there are some requirements:

  • Non-personalized content: The way this service is envisioned to be used is for content that does not change between users. This is imperative.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.

Docs & Support

The easiest way to get started with API Buf.fr is to give it a go! Unfortunately, though, it is hard to see the effect of API Buf.fr easily.
